# Changelog

## Purpose

Track API changes that impact client integrations, including behavior changes, deprecations, and new endpoint groups.

## Entry format

- Date
- Scope (`public`, `customer`, `admin`, `super-admin`, `internal`)
- Change type (`added`, `changed`, `deprecated`, `removed`, `fixed`)
- Migration guidance
