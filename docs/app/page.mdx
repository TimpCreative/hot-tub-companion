# Hot Tub Companion Documentation

This documentation is the primary technical writing surface for all API behavior and platform workflows.

## What this site contains

- End-to-end onboarding for new engineers
- Core architecture and data ownership concepts
- Authentication and tenant-scoping rules
- Full API reference for every endpoint in the platform
- Change management guidance and release notes

## Fast path

1. Read **Getting Started**
2. Review **Core Concepts**
3. Use **API Reference** for endpoint-level contracts and examples
4. Use **Internal Explorer** for operational/debugging workflows

## Endpoint coverage model

Every operation is sourced from generated route inventory and OpenAPI artifacts, then paired with technical writing metadata so reference pages include purpose, constraints, and usage context.
